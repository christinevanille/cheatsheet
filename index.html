Colors

Ways Property Value Declaration Description Ranges
Color Name Keywords Color Pink color: pink; Color name keywords can be used to set color property values for elements in CSS. There are 147 named colors, which you can review here.
Hexadecimal Colors Color #F1A9BB color: #F1A9BB; Hex values always begin with # and specify values of red, blue and green using hexademical numbers such as #F1A9BB. Hexadecimal is a number system with 16 digits, from 0 to F (0-9 / A-F).
rgb() Colors Color rgb(219, 134, 169) color: rgb(219, 134, 169); rgb() should be supplied with three values representing red, green, and blue. These values range can from 0 to 255.
HSL Colors Color hsl(351, 100%, 72%) color: hsl(351, 100%, 72%); This syntax contains three values: hue (the color value itself), saturation (intensity), and lightness. Hue values range from 0 to 360 while saturation and lightness values are represented as percentages.
Color Alpha Values Color rgba(106, 201, 197, 0.5); color: rgba(106, 201, 197, 0.5); Alpha values can be set for RGB colors by using rgba() and providing a fourth value representing alpha. Alpha values determine the transparency of colors in CSS. Alpha values can range between 0.0 (totally transparent) and 1.0 (totally opaque).
Color hsla(177, 47%, 60%, 1) color: hsla(177, 47%, 60%, 1); Alpha values can be set for HSL colors by using hsla() and providing a fourth value representing alpha.
Color Transparent color: transparent; The CSS transparent value can also be used to create a fully transparent element.
